# Astro Salvager - Vertical Slice

## Instalación Rápida

```bash
# 1. Instalar dependencias
npm install

# 2. Ejecutar en modo desarrollo
npm run dev

# El juego se abrirá en http://localhost:5173
```

## Controles
- **WASD** o **Flechas**: Mover nave
- **R**: Reiniciar (después de Game Over)

## Objetivo
Recolecta las piezas de chatarra espacial (ovnis) sin chocar con los asteroides.
