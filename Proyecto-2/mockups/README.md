# Mockups y Wireframes

Esta carpeta contiene los prototipos visuales de la interfaz de usuario del juego.

## Estructura

```
mockups/
├── ui/                     # Interfaces de usuario
│   ├── hud_ingame.png      # HUD durante gameplay
│   ├── main_menu.png       # Menú principal
│   ├── pause_menu.png      # Menú de pausa
│   └── inventory.png       # Pantalla de inventario
├── wireframes/             # Bocetos de baja fidelidad
│   └── ui_flow.png         # Flujo entre pantallas
└── prototipos/             # Prototipos interactivos (si aplica)
```

## Herramientas Recomendadas

### Para Mockups de Alta Fidelidad
- **Figma** (https://figma.com) - Recomendado, gratis
- **Adobe XD** (gratis para uso básico)
- **Sketch** (solo Mac)

### Para Wireframes de Baja Fidelidad
- **Balsamiq** (estilo boceto)
- **Excalidraw** (https://excalidraw.com) - Gratis, online
- **Papel y lápiz** → Escanear/fotografiar

### Para Prototipos Interactivos
- **Figma** (tiene prototipado interactivo)
- **InVision**
- **ProtoPie**

## Checklist de Mockups Requeridos

### HUD In-Game
- [ ] Barra de vida
- [ ] Barra de stamina/mana
- [ ] Indicador de munición/recursos
- [ ] Minimapa
- [ ] Hotbar de items
- [ ] Marcadores de objetivo

### Menú Principal
- [ ] Logo del juego
- [ ] Botones principales (New Game, Continue, Options, Exit)
- [ ] Fondo/animación
- [ ] Versión del juego

### Menú de Pausa
- [ ] Opciones (Resume, Settings, Quit)
- [ ] Blur del gameplay de fondo

### Inventario
- [ ] Grid de items
- [ ] Panel de descripción de item seleccionado
- [ ] Stats del jugador
- [ ] Equipamiento

### Pantalla de Opciones
- [ ] Sliders de volumen
- [ ] Configuración de gráficos
- [ ] Controles/Keybindings
- [ ] Botones Apply/Cancel

## Formato de Entrega

- **Resolución:** 1920x1080 (Full HD)
- **Formato:** PNG con transparencia (si aplica) o JPG
- **Nomenclatura:** `[tipo]_[nombre].png` (ej: `ui_main_menu.png`)

## Anotaciones

Si los mockups tienen interacciones complejas, añade anotaciones:
- Flechas indicando flujo
- Notas sobre comportamiento al hover/click
- Animaciones esperadas
