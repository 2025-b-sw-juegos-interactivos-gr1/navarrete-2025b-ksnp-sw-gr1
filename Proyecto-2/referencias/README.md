# Referencias Visuales y de Audio

Esta carpeta contiene todo el material de referencia utilizado para el diseño del juego.

## Estructura

```
referencias/
├── visual/                 # Referencias visuales
│   ├── moodboard.png       # Moodboard principal
│   ├── personajes/         # Referencias de personajes
│   ├── ambientes/          # Referencias de escenarios
│   ├── ui/                 # Referencias de interfaces
│   └── efectos/            # Referencias de VFX
├── audio/                  # Referencias de audio
│   ├── musica_playlist.md  # Lista de tracks de referencia
│   └── sfx_references.md   # Referencias de efectos de sonido
└── juegos/                 # Análisis de juegos similares
    └── analisis_[juego].md # Análisis detallado por juego
```

## Cómo Organizar Referencias

### 1. Moodboard Principal

Crea un collage visual con:
- 30-50 imágenes de referencia
- Categorías claramente separadas
- Anotaciones de qué te gusta de cada imagen

**Herramientas:**
- **PureRef** (https://www.pureref.com/) - App dedicada para referencias
- **Figma** - Para moodboards colaborativos
- **Pinterest** - Para recolectar imágenes
- **Milanote** - Tablero visual

### 2. Referencias por Categoría

Organiza las imágenes en subcarpetas:

#### Personajes (`personajes/`)
- Diseño de protagonista
- Diseño de enemigos
- Poses y siluetas
- Expresiones faciales

#### Ambientes (`ambientes/`)
- Arquitectura
- Naturaleza
- Iluminación
- Composición de escena

#### UI (`ui/`)
- Layouts de HUD
- Estilos de menú
- Iconografía
- Tipografía

#### Efectos (`efectos/`)
- VFX (explosiones, magia, etc.)
- Partículas
- Animaciones de impacto

### 3. Referencias de Audio

#### Playlist Musical
Crear archivo `audio/musica_playlist.md`:

```markdown
# Playlist de Referencias Musicales

## Enlaces a Playlists
- Spotify: [URL]
- YouTube Music: [URL]

## Tracks Principales

1. **[Nombre del Track]** - [Artista]
   - Juego/Soundtrack: [Origen]
   - Qué tomamos: [Ej: Instrumentación, tempo, mood]
   - Timestamp relevante: [2:30 - 3:15]

2. **[Track 2]**
   - ...

[10-15 tracks total]
```

#### Referencias de SFX
Crear archivo `audio/sfx_references.md`:

```markdown
# Referencias de Efectos de Sonido

## Juegos de Referencia
- [Juego 1]: Excelentes SFX de combate
- [Juego 2]: Ambiente inmersivo

## SFX Específicos
- **Pasos:** Referencia - The Last of Us (natural, pesado)
- **Espadas:** Referencia - Ghost of Tsushima (impacto satisfactorio)
- **UI:** Referencia - Persona 5 (estiloso, memorable)
```

### 4. Análisis de Juegos Similares

Para cada juego de referencia, crear un análisis:

**Plantilla:** `juegos/analisis_[nombre_juego].md`

```markdown
# Análisis: [Nombre del Juego]

## Información General
- Desarrollador: [Nombre]
- Año: [Año]
- Plataforma: [PC, Console, etc.]
- Género: [Género]

## Análisis Visual

### Estilo Artístico
- [Descripción del estilo]

### Lo que funciona ✅
- [Lista de aspectos positivos]

### Lo que evitar ❌
- [Lista de aspectos negativos o no deseados]

### Capturas de Pantalla
- [Ubicación de screenshots]

## Análisis de Gameplay

### Mecánicas Destacadas
- [Mecánicas que nos inspiraron]

### Sentimiento (Feel)
- [Cómo se siente jugar]

## Análisis de Audio

### Música
- [Estilo musical, efectividad]

### SFX
- [Efectos de sonido memorables]

## Lecciones Aprendidas

### Qué tomaremos para nuestro juego
- [Lista]

### Qué haremos diferente
- [Lista]
```

## Fuentes de Referencias

### Imágenes
- **ArtStation** - https://www.artstation.com/
- **Pinterest** - Tableros de game art
- **Behance** - https://www.behance.net/
- **DeviantArt** - https://www.deviantart.com/
- **Game UI Database** - https://www.gameuidatabase.com/

### Audio
- **Freesound** - https://freesound.org/ (SFX gratis)
- **YouTube** - Soundtracks oficiales
- **Spotify** - Playlists de OSTs
- **SoundCloud** - Música indie

### Análisis de Juegos
- **YouTube** - Análisis y reviews
- **GDC Talks** - Postmortems de desarrolladores
- **Gamasutra** - Artículos de desarrollo

## Convenciones de Nombres de Archivos

```
Formato: [categoria]_[descripcion]_[numero].[ext]

Ejemplos:
- personaje_protagonista_ref_01.jpg
- ambiente_bosque_noche_03.png
- ui_hud_futurista_02.jpg
- vfx_explosion_magica_01.gif
```

## Checklist de Referencias

- [ ] Moodboard principal creado (mínimo 30 imágenes)
- [ ] Referencias de personajes (10+ imágenes)
- [ ] Referencias de ambientes (10+ imágenes)
- [ ] Referencias de UI (5+ imágenes)
- [ ] Playlist musical (10+ tracks)
- [ ] Al menos 3 análisis de juegos similares
- [ ] Todas las imágenes organizadas en subcarpetas
- [ ] Créditos y fuentes documentadas
